# TLB EAT 計算題 — 評分準則

## 題目
In a system with a TLB hit ratio of 85%, if TLB access time is 15ns and memory access time is 100ns, calculate the effective memory access time (EAT).

## 滿分：2 分

## 參考答案
EAT = 0.85 × (15 + 100) + 0.15 × (15 + 100 + 100) = 0.85 × 115 + 0.15 × 215 = 97.75 + 32.25 = **130 ns**

## 評分標準

### 2 分（滿分）— 正確
- 答案為 130 或 130ns（含 130.00、130n 等輕微筆誤），或
- 公式完整正確且可算出 130，無論是否寫出最終數值

### 1 分（部分正確）— 有正確概念但未完全正確
- 公式架構正確（兩段式：hit + miss），但有算術錯誤導致答案非 130
  - 例：127.75（miss 部分漏加第二次 TLB access 15ns）
  - 例：131.5、130.22（計算誤差）
  - 例：117.25（miss penalty 用錯數值）
  - 例：124、102 等接近但不正確的數值
- 公式完全正確，但未算出最終數值（如僅寫公式）

### 0 分（錯誤）— 不理解或嚴重錯誤
- 答案明顯錯誤且公式不正確或未呈現正確概念
  - 例：30、30ns、20ns、45ns、100ns、200、1.3、42.75ns、205.25ns、27.75ns、110.5ns 等
- 僅計算 TLB 部分，完全忽略 memory access 部分
- 公式結構錯誤（如：0.85×15 + 0.15×200）

## Reason 政策
- 滿分（2 分）→ reason 留空
- 非滿分（0 或 1 分）→ 必填簡短扣分原因
