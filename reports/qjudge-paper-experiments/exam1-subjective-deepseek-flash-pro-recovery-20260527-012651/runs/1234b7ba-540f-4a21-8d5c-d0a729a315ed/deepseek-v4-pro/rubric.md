# DMA (Direct Memory Access) 評分準則

**題目**：Please briefly describe the Direct Memory Access (DMA)?
**滿分**：2 分
**題型**：essay（短答/申論）

---

## 評分要點

參考答案核心要素：
1. DMA 是一種讓 devices 直接對主記憶體讀寫／傳輸資料的技術
2. 不需要 CPU 介入每次傳輸（bypass CPU）
3. 好處：降低 CPU 使用率／提升系統效能／節省 CPU 時間／高速大量傳輸

## 給分標準

### 2 分（滿分）
- 正確指出 DMA 讓 devices 可直接存取 memory，**且**
- 明確提到 **不須 CPU 介入/干預**，**且**
- 至少提及一項優點（降低 CPU 使用率、提升效能、節省 CPU 時間、高效率傳輸等）
- 可接受同義詞：involve/intervene/help/intervention/without CPU

### 1 分
- 掌握基本概念（devices 直接存取 memory），但：
  - 未明確提及不須 CPU 介入，或
  - 未提及任何好處，或
  - 表述過於簡略／模糊但仍可辨識核心概念，或
  - 有小錯誤但整體方向正確（例：說 DMA 是 device/hardware 而非 technique 不扣分）

### 0 分
- 完全錯誤、答非所問、無意義回答
- 與 DMA 概念無關（例如：只談 virtual memory/paging、只談 I/O interrupt 流程但扭曲 DMA 角色）
- 僅寫出全名 "Direct Memory Access" 而無任何解釋
