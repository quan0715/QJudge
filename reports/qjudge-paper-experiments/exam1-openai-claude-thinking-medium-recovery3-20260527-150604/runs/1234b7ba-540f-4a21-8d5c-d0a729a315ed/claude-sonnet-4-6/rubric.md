# Rubric：Direct Memory Access (DMA)

**題目**：Please briefly describe the Direct Memory Access (DMA)?
**滿分**：2 分
**題型**：essay（申論/短答）

---

## 評分標準

| 分數 | 判準 |
|------|------|
| 2 | 正確說明 DMA 的核心概念：裝置可**不經 CPU 介入**直接與主記憶體交換資料，並提及其目的或效益（如減少 CPU 占用、提升系統效能、加速大量資料傳輸等）之一。 |
| 1 | 僅片面提及 DMA（如只說「繞過 CPU」而未解釋傳輸對象/目的，或只說「裝置傳資料給記憶體」而未提及 CPU 不介入），概念方向正確但說明不完整。 |
| 0 | 完全錯誤、空白、或與 DMA 無關。 |

## 參考要點（任兩項均可得滿分）
1. DMA 是一種資料傳輸技術，允許裝置（硬碟、音效卡、網卡等）直接讀寫主記憶體。
2. 傳輸過程**不需要 CPU 逐步介入**（CPU 不參與每次傳輸）。
3. 可減少 CPU 占用 / 讓 CPU 去做其他工作。
4. 提升系統效能或大量資料的傳輸速度。
5. 由 DMA controller 獨立控制記憶體存取。
