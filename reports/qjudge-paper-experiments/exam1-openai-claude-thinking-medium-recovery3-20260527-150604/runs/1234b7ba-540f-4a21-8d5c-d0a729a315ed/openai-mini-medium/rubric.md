# DMA 申論題評分準則

- 題目：請簡述 Direct Memory Access（DMA）
- 滿分：2 分

## 給分重點

1. **核心定義正確（1 分）**
   - 需說明 DMA 是一種讓裝置與主記憶體直接傳輸資料的技術。
   - 可接受描述：資料傳輸不需 CPU 每次都介入、由 DMA controller 協助搬移資料。

2. **效果或優點正確（1 分）**
   - 需提到可減少 CPU 負擔 / 釋放 CPU 時間 / 提升效率或傳輸速度。
   - 若只寫「快速」但未說明與 CPU 或記憶體傳輸關係，視情況酌扣。

## 扣分原則

- 只提到記憶體、I/O、速度等零散關鍵字但沒有 DMA 核心概念：0–1 分。
- 將 DMA 誤解為 CPU 直接搬資料、或概念明顯錯誤：0 分。
- 答案完整但用詞不同，只要意思正確即可給滿分。
