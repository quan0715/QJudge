# 批改準則

- 題目滿分：2 分
- 題意：解釋 Belady’s Anomaly，並指出有此現象的頁面置換演算法

## 參考答案重點
1. Belady’s Anomaly：頁框數增加，頁面缺失次數反而增加。
2. 發生於 FIFO 頁面置換演算法。

## 評分規則
- 2 分：
  - 正確說明 Belady’s Anomaly 的核心定義：增加 frame 反而可能增加 page faults；
  - 且正確指出 FIFO。
- 1 分：
  - 只答對其中一部分：
    - 正確說明現象，但未指出 FIFO；或
    - 指出 FIFO，但對現象描述不完整或不夠精確。
- 0 分：
  - 答非所問、完全錯誤、或無有效作答。

## 批改原則
- 以答案內容是否明確、完整為準；同義表述可接受。
- 只要明確提到「frame 數增加但 page fault 變多」即可視為現象正確。
- 若僅提到 LRU/OPT 不會有此現象，未回答 FIFO，仍屬部分得分或 0 分，視是否有明確指出正確演算法而定。
