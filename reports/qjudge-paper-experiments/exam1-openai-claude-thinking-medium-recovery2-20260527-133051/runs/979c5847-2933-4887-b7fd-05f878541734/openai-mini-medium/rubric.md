# 批改準則

- 題目：TLB 命中率 85%，TLB 存取 15ns，記憶體存取 100ns，求 effective memory access time (EAT)
- 滿分：2 分
- 參考答案：
  - 命中：15 + 100 = 115ns
  - 未命中：15 + 100 + 100 = 215ns
  - EAT = 0.85 × 115 + 0.15 × 215 = 130ns

## 給分標準
- 2 分：
  - 算式正確，且最後答案為 130ns（單位可寫 ns）
- 1 分：
  - 有正確列出部分公式/加權概念，或中間步驟大致正確，但最後數值有小錯
- 0 分：
  - 公式、概念或計算大幅錯誤，無法得到正確 EAT

## 批改注意
- 只看學生作答內容，不使用既有人工分數或評語
- 若答案只有最終數值，且正確為 130ns，可給滿分
- 若答案有正確方向但未寫完整計算，可依正確程度給 1 分
- 若答案明顯是其他值或與題意無關，給 0 分
