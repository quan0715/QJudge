# 評分 Rubric：SCAN / C-SCAN 磁碟排程

## 題目摘要
- 磁碟 150 個磁軌 (0–149)，磁頭在 80，往增加方向移動
- 請求序列：20, 45, 10, 120, 130, 5, 90
- 滿分：2 分（每小題 1 分）

## 參考答案
- **SCAN**：90, 120, 130, 45, 20, 10, 5
- **C-SCAN**：90, 120, 130, 5, 10, 20, 45

## 評分原則

### 格式寬容
- 分隔符號（逗號、空格、箭頭、頓號）不拘
- 大小寫不拘（scan / SCAN / Scan 均可）
- 包含起始磁軌 (80) 或尾端磁軌 (149/0) 表示路徑 → 不扣分，只要請求的 service order 正確
- 包含排序過程或註解（如「先 sort:...」）→ 不扣分

### SCAN (1 分)
正確順序：90 → 120 → 130 ➔ (反轉) → 45 → 20 → 10 → 5

| 狀況 | 給分 |
|------|------|
| 完全正確 | 1 |
| 順序正確但遺漏 1 個元素 | 0.5 |
| 方向錯誤（如 130 後直接往小走但順序亂） | 0 |
| 誤寫成 C-SCAN 順序（130 後跳 5,10,20,45） | 0 |

### C-SCAN (1 分)
正確順序：90 → 120 → 130 ➔ (跳到 0) → 5 → 10 → 20 → 45

| 狀況 | 給分 |
|------|------|
| 完全正確 | 1 |
| 順序正確但遺漏 1 個元素 | 0.5 |
| 方向錯誤 | 0 |
| 誤寫成 SCAN 順序（130 後反轉 45,20,10,5） | 0 |

### 特殊情況
- SCAN 與 C-SCAN 答案互換 → 每小題各 0 分（兩題皆錯）
- 兩題答案相同 → 至少有一題錯，正確的那題給分，錯誤的 0 分
- 其他不完整或明顯亂寫 → 依正確元素比例給 0 / 0.5 / 1
