# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-04-08

### Added
- Classroom management: create classrooms, invite students via join link
- Announcement system: teachers can post announcements within classrooms
- Contest/exam system: create exams and homework with scheduling, scoring, and draft/publish states
- Question JSON import/export for backup and migration
- In-browser IDE for writing and submitting code
- Auto-grading with support for C, C++, Python, Java, and more
- Exam statistics and per-student submission tracking
- Exam proctoring: tab-switch detection and logging
- Classroom overview with calendar-style event timeline
- AI teaching assistant chatbot (experimental)
- Built-in documentation site at `/docs`
- Multi-language UI: zh-TW, en, ja, ko
- Dark/light theme support
