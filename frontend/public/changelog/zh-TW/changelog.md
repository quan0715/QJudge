# 更新紀錄

## v1.0.0 — 2026-04-08

首次正式版本發布。

### 新功能

- **教室管理**：老師可建立班級、透過邀請連結讓學生加入
- **公告系統**：老師可在教室中發佈公告，學生即時收到通知
- **考試系統**：支援建立考試與作業，設定時間、配分與公開/草稿狀態
- **題目匯入匯出**：考試題目支援 JSON 格式匯入匯出，方便備份與搬移
- **線上作答**：學生可在瀏覽器內的 IDE 撰寫程式碼並提交評測
- **自動評測**：支援多種程式語言（C, C++, Python, Java 等），即時回傳評測結果
- **成績統計**：老師可查看考試成績統計圖表與個別學生作答情況
- **考試防弊**：偵測學生切換分頁行為，並記錄於後台供老師查閱
- **教室行事曆**：教室總覽頁面以行事曆形式顯示近期考試與活動
- **AI 助教**：整合 AI 聊天助手，協助學生理解題目與除錯（實驗功能）
- **使用說明文件**：內建完整的使用說明，涵蓋學生、老師與管理員指南
- **多語言介面**：支援繁體中文、English、日本語、한국어
- **深色模式**：全站支援淺色與深色主題切換
