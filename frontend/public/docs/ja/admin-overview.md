# 管理者機能

このドキュメントでは、QJudge の管理者向け機能について説明します。

## ユーザー管理

### ユーザーの管理

管理者はすべてのユーザーを管理できます：

1. システム > ユーザーに移動
2. 検索・フィルターでユーザー一覧を表示
3. 利用可能なアクション：
   - ユーザーロールの変更
   - パスワードのリセット
   - アカウントの有効化/無効化

### ユーザーロール

| ロール  | 権限                           |
| ------- | ------------------------------ |
| Student | 問題の閲覧、コンテストへの参加 |
| Teacher | コンテストの作成、問題の管理   |
| Admin   | システム全体へのアクセス       |

## システム設定

### 環境設定

システム全体の設定：

- ジャッジサーバー設定
- ファイルアップロード制限
- セッションタイムアウト
- 機能トグル

### お知らせ

システムお知らせの作成・管理：

1. 管理 > お知らせに移動
2. 新しいお知らせを作成
3. 公開設定とタイミングを設定

## 問題バンク

### グローバル問題管理

問題バンク全体の管理：

- 提出された問題のレビュー
- 問題の公開設定
- 問題カテゴリの管理

## セキュリティ

### アクセス制御

システムセキュリティの管理：

- ログイン試行の確認
- IP ブロック
- セッション管理
