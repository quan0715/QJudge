## QJudge とは？

QJudge は、プログラミング教育と練習のために設計されたオンラインジャッジシステムです。以下のことができます：

- **問題を解く** - さまざまな難易度のプログラミング問題を解く
- **コンテストに参加** - 他の参加者と競い合う
- **進捗を追跡** - 解答統計と提出履歴を確認

## 主な機能

### 問題演習

システムは、さまざまな難易度レベルの問題を提供しています：

| 難易度 | 説明                                           |
| ------ | ---------------------------------------------- |
| Easy   | 初心者向け、基本的な構文練習                   |
| Medium | プログラミング基礎が必要、一般的なアルゴリズム |
| Hard   | 上級問題、深いアルゴリズム知識が必要           |

### オンラインコンテスト

- リアルタイムランキングシステム
- 複数の採点モードをサポート（ACM/ICPC、OI）
- コンテスト中の Q&A 機能（Clarifications）

### 即時ジャッジ

コードを提出すると、システムが自動的にコンパイルしてテストし、結果をリアルタイムで報告します。

## ユーザーロールと権限

QJudge は複数のユーザーロールをサポートし、それぞれ異なる機能を持っています：

| ロール  | 利用可能な機能                               | 詳細                                           |
| ------- | -------------------------------------------- | ---------------------------------------------- |
| Student | 問題閲覧、コード提出、コンテスト参加         | デフォルトロール                               |
| Teacher | コンテスト作成、問題管理、学生スコア表示     | [教師ガイドを見る](/docs/teacher-overview)     |
| Admin   | ユーザー管理、システム設定、権限管理         | [管理者ガイドを見る](/docs/admin-overview)     |

> **ヒント**：Teacher または Admin ロールへのアップグレードが必要な場合は、システム管理者にお問い合わせください。
