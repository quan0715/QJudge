# プラットフォーム概要

**QJudge** - NYCU Online Judge システムへようこそ！

## QJudge とは？

QJudge は、プログラミング教育と練習のために設計されたオンラインジャッジシステムです。以下のことができます：

- **問題を解く** - さまざまな難易度のプログラミング問題を解く
- **コンテストに参加** - 他の参加者と競い合う
- **進捗を追跡** - 解答統計と提出履歴を確認

## 主な機能

### 問題演習

システムは、さまざまな難易度レベルの問題を提供しています：

| 難易度 | 説明                                           |
| ------ | ---------------------------------------------- |
| Easy   | 初心者向け、基本的な構文練習                   |
| Medium | プログラミング基礎が必要、一般的なアルゴリズム |
| Hard   | 上級問題、深いアルゴリズム知識が必要           |

### オンラインコンテスト

- リアルタイムランキングシステム
- 複数の採点モードをサポート（ACM/ICPC、OI）
- コンテスト中の Q&A 機能（Clarifications）

### 即時ジャッジ

コードを提出すると、システムが自動的にコンパイルしてテストし、結果をリアルタイムで報告します。

## ユーザーロール

QJudge は複数のユーザーロールをサポート：

| ロール  | 説明                                   |
| ------- | -------------------------------------- |
| Student | 一般ユーザー、問題閲覧とコンテスト参加 |
| Teacher | コンテスト作成、問題管理               |
| Admin   | システム全体のアクセスとユーザー管理   |
