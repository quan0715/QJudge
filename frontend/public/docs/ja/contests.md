# コンテストに参加

QJudge でプログラミングコンテストに参加する方法を学びます。

## コンテストの種類

### 公開コンテスト

ログインしたすべてのユーザーが参加できるコンテスト。

### 非公開コンテスト

パスワードが必要なコンテスト。通常、授業の課題や特定のイベントに使用されます。

## 参加方法

### 1. コンテストを見つける

「**コンテスト**」ページで利用可能なコンテストを閲覧します。

### 2. コンテストに入る

コンテストをクリックしてコンテストページに入ります。非公開コンテストの場合はパスワードを入力します。

### 3. ルールを読む

コンテスト開始前に、ルールと採点方法をよく読んでください。

### 4. 問題を解く

コンテストが始まったら：

1. 問題リストから問題を選択
2. 問題の説明を読む
3. コードを書いて提出
4. リアルタイムランキングを確認

## 採点方式

### ACM/ICPC モード

- 解いた問題数で順位を決定
- 同点の場合はペナルティ時間で順位を決定
- 誤答ごとに 20 分のペナルティ

### OI モード

- 通過したテストケースの割合で得点
- 最終スコアは全問題の得点合計

## コンテスト機能

### Clarification（質問）

問題について質問がある場合は、Clarification システムを通じて出題者に質問できます。

### リアルタイムランキング

自分の順位を確認し、他の参加者と比較できます。

## 注意事項

**重要な注意点**：

- コンテスト中はコードやソリューションを共有しないでください
- 学術的誠実性のガイドラインに従ってください
- 他の人の解答はコンテスト終了後にのみ閲覧できます
