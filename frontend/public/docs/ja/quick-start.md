# クイックスタート

このガイドでは、QJudge システムの使い方を説明します。

## ステップ 1：ログイン

以下の方法でログインできます：

- **SSO ログイン** - NYCU アカウントで直接ログイン
- **通常ログイン** - 登録したメールアドレスとパスワードでログイン

## ステップ 2：問題を閲覧

ログイン後、ナビゲーションバーの「**問題**」をクリックして問題一覧ページに移動します。

できること：

- 難易度で問題をフィルタリング
- 特定の問題を検索
- 問題の正答率を確認

## ステップ 3：解答と提出

1. 問題をクリックして問題詳細ページに移動
2. 問題の説明、入出力形式を読む
3. 右側のコードエディタで解答を書く
4. プログラミング言語を選択
5. 「**提出**」ボタンをクリック

## ステップ 4：結果を確認

提出後、システムが自動的にコードをジャッジします。「**提出履歴**」ページで結果を確認できます：

| 結果                        | 説明                 |
| --------------------------- | -------------------- |
| Accepted (AC)               | すべてのテストに合格 |
| Wrong Answer (WA)           | 出力が正しくない     |
| Time Limit Exceeded (TLE)   | 実行時間超過         |
| Memory Limit Exceeded (MLE) | メモリ制限超過       |
| Runtime Error (RE)          | 実行時エラー         |
| Compilation Error (CE)      | コンパイルエラー     |

## ヒント

提出前に、サンプルテストデータでローカルでプログラムをテストすることをお勧めします。

コーディングを楽しんでください！
