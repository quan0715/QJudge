# クイックスタート

このガイドでは、**5分以内**に最初の提出を完了できるように説明します。

## クイックスタートチェックリスト

以下の手順に従って最初の提出を完了してください：

| ステップ | タスク                   | 詳細                                                         |
| -------- | ------------------------ | ------------------------------------------------------------ |
| 1        | QJudge にログイン        | [ステップ 1：ログイン](#ステップ-1ログイン)                  |
| 2        | 適切な問題を見つける     | [ステップ 2：問題を閲覧](#ステップ-2問題を閲覧)              |
| 3        | コードを書いて提出       | [ステップ 3：解答と提出](#ステップ-3解答と提出)              |
| 4        | 評価結果を確認           | [ステップ 4：結果を確認](#ステップ-4結果を確認)              |

---

## ステップ 1：ログイン

以下の方法でログインできます：

| ログイン方法     | 説明                                   | 推奨対象         |
| ---------------- | -------------------------------------- | ---------------- |
| **SSO ログイン** | NYCU アカウントで直接ログイン          | NYCU 学生・教職員|
| **通常ログイン** | 登録したメールアドレスとパスワードでログイン | 外部ユーザー     |

> **初めて SSO を使用する場合？** システムが自動的にアカウントを作成するため、別途登録は不要です。

---

## ステップ 2：問題を閲覧

ログイン後、ナビゲーションバーの「**問題**」をクリックして問題一覧ページに移動します。

### フィルタと検索機能

| 機能             | 使い方                                   |
| ---------------- | ---------------------------------------- |
| 難易度でフィルタ | Easy/Medium/Hard タグをクリック          |
| キーワード検索   | 検索ボックスに問題名または ID を入力     |
| 正答率を表示     | 各問題の右側に AC 率が表示される         |

> **初心者へのヒント**：`Easy` マークの問題から始めましょう。

---

## ステップ 3：解答と提出

```
1. 問題をクリック → 詳細ページへ
        ↓
2. 説明と例を読む
        ↓
3. エディタでコードを書く
        ↓
4. 言語を選択 (C/C++/Python/Java)
        ↓
5. 「提出」ボタンをクリック
```

### 対応言語

| 言語   | バージョン | コンパイル/実行コマンド |
| ------ | ---------- | ----------------------- |
| C      | GCC 11     | `gcc -O2 -std=c11`      |
| C++    | G++ 11     | `g++ -O2 -std=c++17`    |
| Python | 3.10       | `python3`               |
| Java   | 17         | `javac` + `java`        |

詳細は [対応言語](/docs/supported-languages) をご覧ください。

---

## ステップ 4：結果を確認

提出後、システムが自動的にコードを評価します。結果は数秒以内に表示されます。

### 評価結果の説明

| 結果                  | 略称 | 説明                   | 次のアクション                       |
| --------------------- | ---- | ---------------------- | ------------------------------------ |
| Accepted              | AC   | すべてのテストに合格   | おめでとう！次の問題に挑戦           |
| Wrong Answer          | WA   | 出力が正しくない       | ロジックとエッジケースを確認         |
| Time Limit Exceeded   | TLE  | 実行時間超過           | アルゴリズムの効率を最適化           |
| Memory Limit Exceeded | MLE  | メモリ制限超過         | データ構造の使用量を削減             |
| Runtime Error         | RE   | 実行中にクラッシュ     | 配列範囲外アクセス、ゼロ除算を確認   |
| Compilation Error     | CE   | コンパイル失敗         | 構文エラーを確認                     |

詳細なトラブルシューティングは [よくあるエラー](/docs/common-errors) をご覧ください。

---
