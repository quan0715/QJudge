# 教師機能

このドキュメントでは、QJudge の教師向け機能について説明します。

## コンテスト管理

### コンテストの作成

教師はコンテストを作成・管理できます：

1. コンテストページに移動
2. 「コンテストを作成」をクリック
3. コンテスト詳細を入力：
   - コンテスト名と説明
   - 開始・終了時間
   - 採点モード（ACM/ICPC または OI）
   - 公開設定

### 問題の追加

コンテストに問題を追加：

1. コンテスト設定に移動
2. 「問題を管理」タブを選択
3. 問題バンクから問題を検索・追加
4. 配点と順序を設定

### 参加者の管理

参加者を制御：

- 公開コンテスト：すべてのユーザーに公開
- 非公開コンテスト：パスワードが必要
- 招待制：特定のユーザーを追加

## 問題管理

### 問題の作成

教師は新しい問題を作成できます：

1. 問題管理に移動
2. 「問題を作成」をクリック
3. 入力項目：
   - 問題タイトルと説明
   - 入出力形式
   - 制約
   - テストケース

### テストケース

自動採点用のテストケースをアップロード：

- 入力ファイル（.in）
- 期待出力ファイル（.out）
- 時間・メモリ制限を設定

## モニタリング

### リアルタイムダッシュボード

コンテスト中の監視：

- 提出統計
- 問題解決率
- 参加者の進捗

### Clarifications

Clarification システムを通じて参加者の質問に回答。

## レポート

コンテスト後の確認：

- 最終順位
- 提出履歴
- 問題統計
