# 교사 기능

이 문서에서는 QJudge의 교사용 기능에 대해 설명합니다.

## 대회 관리

### 대회 생성

교사는 대회를 생성하고 관리할 수 있습니다:

1. 대회 페이지로 이동
2. "대회 만들기" 클릭
3. 대회 세부 정보 입력:
   - 대회 이름 및 설명
   - 시작 및 종료 시간
   - 채점 모드 (ACM/ICPC 또는 OI)
   - 공개 설정

### 문제 추가

대회에 문제 추가:

1. 대회 설정으로 이동
2. "문제 관리" 탭 선택
3. 문제 은행에서 문제 검색 및 추가
4. 배점 및 순서 설정

### 참가자 관리

참가자 제어:

- 공개 대회: 모든 사용자에게 공개
- 비공개 대회: 비밀번호 필요
- 초대제: 특정 사용자 추가

## 문제 관리

### 문제 생성

교사는 새 문제를 생성할 수 있습니다:

1. 문제 관리로 이동
2. "문제 만들기" 클릭
3. 입력 항목:
   - 문제 제목 및 설명
   - 입출력 형식
   - 제약 조건
   - 테스트 케이스

### 테스트 케이스

자동 채점을 위한 테스트 케이스 업로드:

- 입력 파일 (.in)
- 예상 출력 파일 (.out)
- 시간 및 메모리 제한 설정

## 모니터링

### 실시간 대시보드

대회 중 모니터링:

- 제출 통계
- 문제 해결률
- 참가자 진행 상황

### Clarifications

Clarification 시스템을 통해 참가자 질문에 답변.

## 보고서

대회 후 확인:

- 최종 순위
- 제출 기록
- 문제 통계
