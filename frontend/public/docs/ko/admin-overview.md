# 관리자 기능

이 문서에서는 QJudge의 관리자용 기능에 대해 설명합니다.

## 사용자 관리

### 사용자 관리

관리자는 모든 사용자를 관리할 수 있습니다:

1. 시스템 > 사용자로 이동
2. 검색 및 필터로 사용자 목록 표시
3. 사용 가능한 작업:
   - 사용자 역할 변경
   - 비밀번호 재설정
   - 계정 활성화/비활성화

### 사용자 역할

| 역할    | 권한                 |
| ------- | -------------------- |
| Student | 문제 보기, 대회 참가 |
| Teacher | 대회 생성, 문제 관리 |
| Admin   | 전체 시스템 접근     |

## 시스템 설정

### 환경 구성

시스템 전체 설정:

- 채점 서버 설정
- 파일 업로드 제한
- 세션 타임아웃
- 기능 토글

### 공지사항

시스템 공지사항 생성 및 관리:

1. 관리 > 공지사항으로 이동
2. 새 공지사항 생성
3. 공개 설정 및 타이밍 설정

## 문제 은행

### 전체 문제 관리

전체 문제 은행 관리:

- 제출된 문제 검토
- 문제 공개 설정
- 문제 카테고리 관리

## 보안

### 접근 제어

시스템 보안 관리:

- 로그인 시도 확인
- IP 차단
- 세션 관리
