# 관리자 기능

이 문서에서는 QJudge의 관리자용 기능에 대해 설명합니다.

## 사용자 관리

### 사용자 관리

관리자는 모든 사용자를 관리할 수 있습니다:

1. 시스템 > 사용자로 이동
2. 검색 및 필터로 사용자 목록 표시
3. 사용 가능한 작업:
   - 사용자 역할 변경
   - 비밀번호 재설정
   - 계정 활성화/비활성화

### 사용자 역할

| 역할    | 권한                 |
| ------- | -------------------- |
| Student | 문제 보기, 대회 참가 |
| Teacher | 대회 생성, 문제 관리 |
| Admin   | 전체 시스템 접근     |

### 일괄 작업

사용자 일괄 가져오기 지원, 다음에 적용:

- 학기 초 학생 계정 추가
- 대규모 사용자 관리

## 시스템 설정

### 환경 구성

시스템 전체 설정:

- 채점 서버 설정
- 파일 업로드 제한
- 세션 타임아웃
- 기능 토글

### 공지사항

시스템 공지사항 생성 및 관리:

1. 관리 > 공지사항으로 이동
2. 새 공지사항 생성
3. 공개 설정 및 타이밍 설정

## 문제 은행

### 전체 문제 관리

전체 문제 은행 관리:

- 제출된 문제 검토
- 문제 공개 설정
- 문제 카테고리 관리

## 데이터 백업

정기적으로 다음을 수행하는 것이 좋습니다:

- 데이터베이스 백업
- 문제 테스트 데이터 백업
- 사용자 제출 코드 백업

## 보안

### 접근 제어

시스템 보안 관리:

- 로그인 시도 확인
- IP 차단
- 세션 관리

## 보안 권장 사항

1. **정기적인 비밀번호 업데이트**: 관리자 계정은 강력한 비밀번호를 사용해야 함
2. **관리자 수 제한**: 필요한 인원에게만 권한 부여
3. **로그인 기록 모니터링**: 비정상적인 로그인 행동에 주의
4. **최소 권한 원칙**: 실제 필요에 따라 권한 할당
