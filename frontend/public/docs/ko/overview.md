## QJudge란?

QJudge는 프로그래밍 교육과 연습을 위해 설계된 온라인 저지 시스템입니다. 다음과 같은 기능을 제공합니다:

- **문제 풀이** - 다양한 난이도의 프로그래밍 문제 해결
- **대회 참가** - 다른 참가자들과 경쟁
- **진행 상황 추적** - 풀이 통계 및 제출 기록 확인

## 주요 기능

### 문제 연습

시스템은 다양한 난이도 수준의 문제를 제공합니다:

| 난이도 | 설명                                    |
| ------ | --------------------------------------- |
| Easy   | 초보자용, 기본 구문 연습                |
| Medium | 프로그래밍 기초 필요, 일반적인 알고리즘 |
| Hard   | 고급 문제, 심화 알고리즘 지식 필요      |

### 온라인 대회

- 실시간 순위 시스템
- 다양한 채점 모드 지원 (ACM/ICPC, OI)
- 대회 중 Q&A 기능 (Clarifications)

### 즉시 채점

코드를 제출하면 시스템이 자동으로 컴파일하고 테스트하여 실시간으로 결과를 보고합니다.

## 사용자 역할 및 권한

QJudge는 여러 사용자 역할을 지원하며, 각각 다른 기능을 사용할 수 있습니다:

| 역할    | 사용 가능한 기능                             | 세부 정보                                     |
| ------- | -------------------------------------------- | --------------------------------------------- |
| Student | 문제 보기, 코드 제출, 대회 참가              | 기본 역할                                     |
| Teacher | 대회 생성, 문제 관리, 학생 점수 보기         | [교사 가이드 보기](/docs/teacher-overview)    |
| Admin   | 사용자 관리, 시스템 설정, 권한 관리          | [관리자 가이드 보기](/docs/admin-overview)    |

> **팁**: Teacher 또는 Admin 역할로 업그레이드하려면 시스템 관리자에게 문의하세요.
