# 대회 참가

QJudge에서 프로그래밍 대회에 참가하는 방법을 알아봅니다.

## 대회 유형

### 공개 대회

로그인한 모든 사용자가 참가할 수 있는 대회입니다.

### 비공개 대회

비밀번호가 필요한 대회로, 일반적으로 수업 과제나 특정 이벤트에 사용됩니다.

## 참가 방법

### 1. 대회 찾기

"**대회**" 페이지에서 이용 가능한 대회를 찾아봅니다.

### 2. 대회 입장

대회를 클릭하여 대회 페이지에 입장합니다. 비공개 대회의 경우 비밀번호를 입력해야 합니다.

### 3. 규칙 읽기

대회 시작 전에 규칙과 채점 방식을 주의 깊게 읽어보세요.

### 4. 문제 풀기

대회가 시작되면:

1. 문제 목록에서 문제 선택
2. 문제 설명 읽기
3. 코드 작성 및 제출
4. 실시간 순위 확인

## 채점 방식

### ACM/ICPC 모드

- 푼 문제 수로 순위 결정
- 동점일 경우 페널티 시간으로 순위 결정
- 오답당 20분 페널티

### OI 모드

- 통과한 테스트 케이스 비율로 점수
- 최종 점수는 모든 문제 점수의 합계

## 주의사항

**중요 안내**:

- 대회 중에는 코드나 솔루션을 공유하지 마세요
- 학업 윤리 지침을 준수하세요
- 다른 사람의 답안은 대회 종료 후에만 볼 수 있습니다
