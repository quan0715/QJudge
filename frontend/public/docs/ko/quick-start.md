# 빠른 시작

이 가이드는 **5분 이내**에 첫 번째 제출을 완료하는 데 도움을 드립니다.

## 빠른 시작 체크리스트

다음 단계에 따라 첫 번째 제출을 완료하세요:

| 단계 | 작업                     | 세부 정보                                                    |
| ---- | ------------------------ | ------------------------------------------------------------ |
| 1    | QJudge에 로그인          | [1단계: 로그인](#1단계-로그인)                               |
| 2    | 적합한 문제 찾기         | [2단계: 문제 찾아보기](#2단계-문제-찾아보기)                 |
| 3    | 코드 작성 및 제출        | [3단계: 풀이 및 제출](#3단계-풀이-및-제출)                   |
| 4    | 평가 결과 확인           | [4단계: 결과 확인](#4단계-결과-확인)                         |

---

## 1단계: 로그인

다음 방법으로 로그인할 수 있습니다:

| 로그인 방법      | 설명                              | 권장 대상          |
| ---------------- | --------------------------------- | ------------------ |
| **SSO 로그인**   | NYCU 계정으로 직접 로그인         | NYCU 학생 및 교직원|
| **일반 로그인**  | 등록된 이메일과 비밀번호로 로그인 | 외부 사용자        |

> **처음 SSO를 사용하시나요?** 시스템이 자동으로 계정을 생성하므로 별도 등록이 필요 없습니다.

---

## 2단계: 문제 찾아보기

로그인 후, 네비게이션 바에서 "**문제**"를 클릭하여 문제 목록 페이지로 이동합니다.

### 필터 및 검색 기능

| 기능             | 사용 방법                               |
| ---------------- | --------------------------------------- |
| 난이도별 필터    | Easy/Medium/Hard 태그 클릭              |
| 키워드 검색      | 검색 상자에 문제 이름 또는 ID 입력      |
| 정답률 보기      | 각 문제 오른쪽에 AC 비율 표시           |

> **초보자 팁**: `Easy`로 표시된 문제부터 시작하세요.

---

## 3단계: 풀이 및 제출

```
1. 문제 클릭 → 상세 페이지로 이동
        ↓
2. 설명과 예제 읽기
        ↓
3. 편집기에서 코드 작성
        ↓
4. 언어 선택 (C/C++/Python/Java)
        ↓
5. "제출" 버튼 클릭
```

### 지원 언어

| 언어   | 버전   | 컴파일/실행 명령       |
| ------ | ------ | ---------------------- |
| C      | GCC 11 | `gcc -O2 -std=c11`     |
| C++    | G++ 11 | `g++ -O2 -std=c++17`   |
| Python | 3.10   | `python3`              |
| Java   | 17     | `javac` + `java`       |

자세한 내용은 [지원 언어](/docs/supported-languages)를 참조하세요.

---

## 4단계: 결과 확인

제출 후, 시스템이 자동으로 코드를 평가합니다. 결과는 몇 초 내에 나타납니다.

### 평가 결과 설명

| 결과                  | 약어 | 설명                   | 다음 조치                            |
| --------------------- | ---- | ---------------------- | ------------------------------------ |
| Accepted              | AC   | 모든 테스트 통과       | 축하합니다! 다음 문제에 도전         |
| Wrong Answer          | WA   | 출력이 올바르지 않음   | 로직과 엣지 케이스 확인              |
| Time Limit Exceeded   | TLE  | 실행 시간 초과         | 알고리즘 효율 최적화                 |
| Memory Limit Exceeded | MLE  | 메모리 제한 초과       | 데이터 구조 사용량 줄이기            |
| Runtime Error         | RE   | 실행 중 크래시         | 배열 범위 초과, 0으로 나누기 확인    |
| Compilation Error     | CE   | 컴파일 실패            | 구문 오류 확인                       |

자세한 문제 해결은 [자주 발생하는 오류](/docs/common-errors)를 참조하세요.
