# 빠른 시작

이 가이드는 QJudge 시스템을 빠르게 시작하는 데 도움이 됩니다.

## 1단계: 로그인

다음 방법으로 로그인할 수 있습니다:

- **SSO 로그인** - NYCU 계정으로 직접 로그인
- **일반 로그인** - 등록된 이메일과 비밀번호로 로그인

## 2단계: 문제 찾아보기

로그인 후, 네비게이션 바에서 "**문제**"를 클릭하여 문제 목록 페이지로 이동합니다.

할 수 있는 것:

- 난이도별 문제 필터링
- 특정 문제 검색
- 문제 정답률 확인

## 3단계: 풀이 및 제출

1. 문제를 클릭하여 문제 상세 페이지로 이동
2. 문제 설명, 입출력 형식 읽기
3. 오른쪽 코드 편집기에서 답안 작성
4. 프로그래밍 언어 선택
5. "**제출**" 버튼 클릭

## 4단계: 결과 확인

제출 후, 시스템이 자동으로 코드를 채점합니다. "**제출 기록**" 페이지에서 결과를 확인할 수 있습니다:

| 결과                        | 설명                 |
| --------------------------- | -------------------- |
| Accepted (AC)               | 모든 테스트 통과     |
| Wrong Answer (WA)           | 출력이 올바르지 않음 |
| Time Limit Exceeded (TLE)   | 실행 시간 초과       |
| Memory Limit Exceeded (MLE) | 메모리 제한 초과     |
| Runtime Error (RE)          | 실행 중 오류 발생    |
| Compilation Error (CE)      | 컴파일 실패          |

## 팁

제출하기 전에 샘플 테스트 데이터로 로컬에서 프로그램을 테스트하는 것이 좋습니다.
