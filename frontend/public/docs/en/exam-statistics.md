This guide explains how to use the statistics function after an exam to review student performance, question difficulty, and overall score distribution.

---

## 1. Entering the Statistics Panel (Exam Statistics)

Click the **"Statistics"** icon in the Admin Dashboard sidebar.

> **Note:** This statistics panel is currently designed specifically for **Paper-like Tests**. For Coding Test statistics, please refer to the other guide.

---

## 2. Overall Data Analysis

The statistics panel provides key information to help teachers understand the class status:

### Left Column: Questions Map
- Lists all question numbers and types.
- **Average Score**: Displays the real-time average score for each question, helping teachers identify questions that were particularly difficult or lacked discrimination.
- **Grading Progress**: Shows "X/Y" representing the proportion of graded responses, helping teachers track review progress.

---

## 3. Item Analysis

Click on any question card, and the right column will display deeper data:

### Objective Question Statistics (Single/Multiple Choice, True/False)
- **Option Distribution**: Displays a bar chart showing the number and proportion of students who chose each option.
- **Correct Answer Marking**:
  - **Green Bar**: Represents the correct answer.
  - **Red Bar**: Indicates an incorrect option chosen by too many students, which may suggest ambiguous wording or a collective misunderstanding.
- **Distractor Analysis**: If an incorrect option is never chosen, it is recommended to adjust it in future exams.

### Subjective Question Review (Short Answer, Essay)
- **Response List**: Lists specific text answers from all students.
- **Quick Browse**: Click a student's name to expand their content and preview the depth of their response without entering the grading interface.
- **Score Marking**: Simultaneously displays the score currently given by the teacher.

---

## 4. Applying Feedback to Teaching

Through statistical data, you can:

1. **Post-exam Review**: Focus on questions with the lowest accuracy (red zone) during exam review.
2. **Adjust Weighting**: If a question was missed by many due to unclear description, scores can be adjusted flexibly in the grading page.
3. **Optimize Question Bank**: Use data to fine-tune question and option text.

---

## 5. Exporting Data

- **Question PDF**: Export a clean test paper for offline storage.
- **Answer PDF**: Export a test paper containing correct answers and explanations.
- **Results Report (JSON)**: Can be used for third-party systems or advanced analysis in Excel.
