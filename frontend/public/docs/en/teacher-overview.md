# Teacher Features

This document provides an overview of teacher-specific features in QJudge.

## Contest Management

### Creating Contests

Teachers can create and manage contests:

1. Navigate to Contests page
2. Click "Create Contest"
3. Fill in contest details:
   - Contest name and description
   - Start and end time
   - Scoring mode (ACM/ICPC or OI)
   - Visibility settings

### Adding Problems

Add problems to your contest:

1. Go to contest settings
2. Select "Manage Problems" tab
3. Search and add problems from the problem bank
4. Set point values and order

### Managing Participants

Control who can participate:

- Public contests: Open to all users
- Private contests: Require password
- Invite-only: Add specific users

## Problem Management

### Creating Problems

Teachers can create new problems:

1. Go to Problem Management
2. Click "Create Problem"
3. Fill in:
   - Problem title and description
   - Input/Output format
   - Constraints
   - Test cases

### Test Cases

Upload test cases for automatic judging:

- Input files (.in)
- Expected output files (.out)
- Set time and memory limits

## Monitoring

### Real-time Dashboard

During contests, monitor:

- Submission statistics
- Problem solve rates
- Participant progress

### Clarifications

Respond to participant questions through the clarification system.

## Reports

After contests, view:

- Final standings
- Submission history
- Problem statistics
