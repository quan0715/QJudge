# Admin Features

This document provides an overview of administrator features in QJudge.

## User Management

### Managing Users

Administrators can manage all users:

1. Navigate to System > Users
2. View user list with search and filter
3. Actions available:
   - Change user role
   - Reset password
   - Disable/enable account

### User Roles

| Role    | Permissions                            |
| ------- | -------------------------------------- |
| Student | View problems, participate in contests |
| Teacher | Create contests, manage problems       |
| Admin   | Full system access                     |

## System Settings

### Environment Configuration

Configure system-wide settings:

- Judge server settings
- File upload limits
- Session timeout
- Feature toggles

### Announcements

Create and manage system announcements:

1. Go to Management > Announcements
2. Create new announcement
3. Set visibility and timing

## Problem Bank

### Global Problem Management

Manage the entire problem bank:

- Review submitted problems
- Set problem visibility
- Manage problem categories

### Importing Problems

Import problems from external sources:

- Support for various formats
- Batch import capability

## Monitoring

### System Health

Monitor system status:

- Server resource usage
- Judge queue status
- Active user sessions

### Logs

View system logs for troubleshooting:

- User activity logs
- Judge logs
- Error logs

## Security

### Access Control

Manage system security:

- Review login attempts
- IP blocking
- Session management
