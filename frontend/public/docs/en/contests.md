# Joining Contests

Learn how to participate in programming contests on QJudge.

## Contest Types

### Public Contests

Contests that all logged-in users can participate in.

### Private Contests

Contests that require a password to join, typically used for course assignments or specific events.

## How to Join

### 1. Find a Contest

Browse available contests on the "**Contests**" page. You can see:

- Contest name and description
- Start and end times
- Contest status (Upcoming / Ongoing / Ended)

### 2. Enter the Contest

Click on a contest to enter the contest page. For private contests, you'll need to enter the password.

### 3. Read the Rules

Before the contest starts, carefully read the contest rules and scoring method.

### 4. Solve Problems

Once the contest starts:

1. Select a problem from the problem list
2. Read the problem description
3. Write and submit your code
4. Check the real-time standings

## Scoring Methods

### Scoring Mode

- Ranking based primarily on total score
- When scores are tied, ranking is determined by number of problems solved
- When both are tied, ranking is determined by total penalty time
- Each wrong submission adds 20 minutes of penalty

### OI Mode

- Each problem is scored based on the percentage of test cases passed
- Final score is the sum of all problem scores

## Contest Features

### Clarifications

If you have questions about a problem, you can ask the problem setters through the clarification system.

### Live Standings

View your ranking and compare with other participants.

## Important Notes

**Important Reminders**:

- Do not share code or solutions during the contest
- Follow academic integrity guidelines
- Solutions from others can only be viewed after the contest ends
