# 管理員功能總覽

本文介紹 QJudge 系統中管理員角色可使用的功能。

## 權限說明

管理員帳號擁有最高權限，包括：

- 所有教師權限
- 使用者管理
- 系統設定
- 公告管理

## 使用者管理

### 管理使用者帳號

進入「系統 > 使用者管理」頁面，您可以：

- 查看所有使用者列表
- 修改使用者角色
- 停用/啟用帳號
- 重設使用者密碼

### 使用者角色

| 角色    | 權限範圍             |
| ------- | -------------------- |
| Student | 基本使用者權限       |
| Teacher | 可建立競賽、管理題目 |
| Admin   | 完整系統管理權限     |

### 批次操作

支援批次匯入使用者，適用於：

- 學期初新增學生帳號
- 大規模使用者管理

## 公告管理

### 發布公告

1. 進入「管理 > 公告管理」
2. 點擊「新增公告」
3. 填寫公告標題與內容
4. 設定顯示狀態

### 公告設定

- **顯示/隱藏**：控制公告是否在首頁顯示
- **置頂**：重要公告可置頂顯示
- **時效**：可設定公告的顯示期間

## 系統設定

### 環境設定

進入「系統 > 環境設定」頁面，可配置：

- 評測系統連線狀態
- 系統參數設定
- 維護模式開關

### 評測系統監控

- 查看評測佇列狀態
- 監控系統資源使用
- 檢視錯誤日誌

## 資料備份

建議定期進行：

- 資料庫備份
- 題目測試資料備份
- 使用者提交程式碼備份

## 安全建議

1. **定期更新密碼**：管理員帳號應使用強密碼
2. **限制管理員數量**：僅授權必要人員
3. **監控登入紀錄**：注意異常登入行為
4. **權限最小化**：依實際需求分配權限
