# 管理員功能總覽（現況）

> 文件狀態：2026-02-24

## 管理員權限範圍

- 使用者與角色管理
- 系統層設定與運維檢查
- 公告與平台治理
- 教師/課程流程支援

## 目前建議巡檢項目

1. 服務可用性：frontend/backend/ai-service/postgres/redis/celery
2. AI 健康檢查：`GET /health`（ai-service）
3. 競賽與考試相關權限設定
4. 重要公告是否有效顯示

## 安全與稽核重點

- 管理員帳號採最小權限原則。
- 定期輪替密碼與檢查登入記錄。
- 涉及考試場景時，確認違規事件記錄機制可運作。

## 維運建議

- 先在 docker dev 環境復現問題，再套用到正式環境。
- 每次調整前保留資料庫備份與操作紀錄。
