# 建立考試與題目

> 文件狀態：2026-03-04

QJudge 支援兩種比賽類型：**Coding Test**（程式題）和 **紙筆題考試**（Paper Exam），並提供防作弊監控與多樣的題型配置。
本篇指南將帶領教師了解如何建立考試、設定防作弊模式，並建立測驗題目 (Exam Questions)。

## 1. 建立一場新競賽

建立時系統會要求您選擇比賽類型：

1. 請前往系統上方的選單或點擊儀表板上的 **我的競賽管理**。
2. 點擊頁面右上角的 **新增競賽** 按鈕。
   ![新增競賽範例圖](/docs/images/create_exam_tutorial.png)
3. 選擇比賽類型：
   - **Coding Test**：程式題為主，支援提交與判題。
   - **紙筆題考試**：考卷題型，支援是非/選擇/問答題。
4. 填寫競賽名稱與基本資訊。
5. 設定考試的 **開始時間** 與 **結束時間**。

## 2. 啟用防作弊模式 (Cheat Detection)

進入競賽編輯頁面的 **設定 (Settings)** 區塊後，您可以啟用防作弊設定。

1. 在設定列表中找到 **防作弊設定 (Cheat Detection Settings)**。
2. 勾選 **啟用防作弊模式** 以強制學生在進入時進行全螢幕鎖定，並開啟切換分頁等違規監測。
   ![啟用防作弊模式](/docs/images/enable_exam_mode.png)
3. 您可以彈性設定：
   - **最大違規警告次數**：若設定為 0，學生違規將會立刻被鎖定作答權限；若設定大於 0（例如 3），學生在達到限制次數前只會收到警告。
   - **允許自動解鎖**：若開啟，可設定學生被鎖定多少分鐘後，系統自動幫其解鎖以繼續測驗。

> **提示：** 強烈建議將實際考試隱藏或設立 **加入密碼** 以限制非預期同學加入；同時可以關閉「比賽期間顯示排行榜」，避免學生於考試進行中看見其他人的進度。

## 3. 建立並編輯考試題目

選擇「紙筆題考試」類型的競賽中可以設定選擇題、是非題等測驗題。這部分可以在 Admin Dashboard 的 **Exam Management** 面板進行編輯。

### 題目編輯器佈局

編輯器採用兩欄設計：

| 區域 | 內容 |
|------|------|
| **左欄：題目樹** | 以列表形式顯示所有題目卡片，支援拖曳排序 |
| **右欄：編輯面板** | 點選題目後顯示完整的編輯表單 |

### 新增題目

1. 點擊左欄底部的 **新增題目** 按鈕，新題目會加到列表末端。
2. 若要在特定位置插入，可點擊題目卡片之間的 **「+」** 按鈕，新題目會插入到該位置。

### 題型選擇

系統支援以下五種題型：

| 題型 | 說明 | 自動批改 |
|------|------|---------|
| 單選題 (Single Choice) | 多個選項，單一正解 | 支援 |
| 多選題 (Multiple Choice) | 多個選項，多個正解 | 支援 |
| 是非題 (True/False) | 固定 True / False 兩選項 | 支援 |
| 簡答題 (Short Answer) | 學生輸入簡短文字 | 不支援 |
| 問答題 (Essay) | 學生長篇敘述回答 | 不支援 |

### 編輯題目內容

1. **題目敘述**：支援 Markdown 及 LaTeX 語法（如 `$x^2$`、`$$\sum_{i=1}^n$$`）。可透過編輯框上方的 **編輯 / 預覽** 切換鍵確認渲染結果。
2. **設定配分**：為每道題設定分數，左欄底部會自動加總顯示「總配分」。
3. **增刪選項**（僅選擇題）：點擊 **新增選項** 輸入選項文字，點擊選項右側的刪除按鈕移除。
4. **設定正確答案**：在下拉式區塊或選項列表中指派正確答案。
5. **參考答案**（僅主觀題）：可選填參考答案，供批改時參照。
6. 完成後點擊 **儲存**。

### 複製題目

在題目卡片上點擊 **複製** 按鈕，系統會在該題下方插入一份完整副本（包含題型、選項、正確答案與配分），方便快速建立相似題目。

## 4. 變更順序與刪除問題

- **拖曳排序**：在題目清單中，按住題目卡片左側的拖曳控制桿 (Drag Handle)，拖拉以調整題目順序。
- **編輯**：點擊題目卡片即可在右欄開啟編輯表單。
- **刪除**：在編輯面板中點擊左下角的 **刪除此題目** 按鈕。

> **注意：** 一旦有學生已經開始作答（題目被凍結），「拖曳排序、編輯及刪除功能」將會被系統鎖定以確保試卷的一致性與公平。凍結狀態下仍可新增題目。

## 5. 匯出與匯入

考試題目支援 JSON 格式的批量匯出與匯入，方便跨考試複製題庫。

- **匯出**：點擊工具列的 **匯出檔案**，選擇匯出格式（試題 PDF / 答案 PDF / Exam JSON）
- **匯入**：點擊 **匯入 JSON**，上傳 `.json` 檔案後系統會進行格式檢查與預覽，確認後覆蓋現有題目

詳細的 JSON 格式規範與匯入規則，請參閱 [Exam 題目 JSON 匯入/匯出教學](/docs/zh-TW/exam-json-import-export)。
