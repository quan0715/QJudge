QJudge 支援兩種主要競賽類型：**Coding Test**（程式開發測驗）與 **紙筆題考試** (Paper-like Test)，本指南將詳細教導如何建立一個專業的線上測驗。

---

## 1. 建立一場新考試 (Paper-like Test)

若要建立類似紙筆測驗的選擇題、是非題 or 問答題考卷，請按照以下步驟：

1. **進入管理介面**：登入後，點擊頂部導覽列或儀表板上的 **「我的競賽管理」**。
2. **新增競賽**：點擊頁面右上角的綠色 **「新增競賽」** 按鈕。
   ![新增競賽範例圖](/docs/images/create_exam_tutorial.png)
3. **選擇類型**：在彈出的視窗中，將「競賽類型」切換為 **「紙筆題考試」(Paper Exam)**。
4. **填寫基本資訊**：
   - **競賽名稱**：例「114學年度期中考」。
   - **開始與結束時間**：設定學生可以進入系統作答的時間區段。
   - **可見性**：建議設為「隱藏」，僅提供密碼或網址給考生；或設為「公開」讓所有人在大廳看到。
5. **設定加入密碼**：若要限制特定班級參加，建議設定「加入密碼」。

---

## 2. 核心設定：防作弊與作答限制

進入競賽編輯頁面後，點擊左側導覽的 **「設定 (Settings)」** 分頁：

### 啟用防作弊模式 (Cheat Detection)
1. 找到 **「防作弊設定」** 區塊，勾選 **「啟用考試模式」**。
   ![啟用考試模式](/docs/images/enable_exam_mode.png)
2. **最大違規次數**：
   - **設定為 0**：學生只要切換分頁或離開全螢幕，系統會立即鎖定作答權限（建議用於嚴格考試）。
   - **設定大於 0**：學生在達到次數前只會收到系統警告。
3. **允許自動解鎖**：
   - 若開啟，可設定學生被鎖定後 **N 分鐘** 系統會自動解鎖。這能減輕監考老師手動解鎖的壓力。

### 公平性設定
- **關閉排行榜**：強烈建議在考試期間關閉「顯示排行榜」，避免學生互相抄襲或看見他人進度。
- **隨機化選項 (開發中)**：確保不同學生的題目順序或選項順序不同（請關注後續更新）。

## 3. 詳細題目編輯教學 (Question Authoring)

點擊 Admin Dashboard 的 **「Exam Management」** 面板，開始編寫試卷內容。

### 編輯介面說明
- **左側列表 (Question Tree)**：管理所有題目。
  - **「+」按鈕**：在特定位置插入新題。
  - **拖曳圖示 (Drag Handle)**：按住左側六點圖示可上下調整順序。
- **右側編輯區 (Editor)**：撰寫單一題目的詳細細節。

### 步驟 A：選擇題型 (Question Type)
QJudge 提供五種核心題型，系統會根據題型自動切換編輯器欄位：

| 題型名稱 | 適用場景 | 自動計分 |
| :--- | :--- | :--- |
| **單選題** | 多個選項，只有一個正確答案 | 是 |
| **多選題** | 多個選項，可以有多個正確答案（全對才給分） | 是 |
| **是非題** | 固定 True / False | 是 |
| **簡答題** | 學生輸入一段簡短文字（例：關鍵字） | 否 (需人工) |
| **問答題** | 學生撰寫長篇敘述（例：實作思路） | 否 (需人工) |

### 步驟 B：撰寫題目內容 (Content)
1. **Markdown 與 LaTeX**：支援完整的排版功能。
   - 使用 `$x^2$` 輸入行內公式。
   - 使用 `$$\sum_{i=1}^n$$` 輸入獨立行公式。
   - 支援貼入程式碼區塊 (Code Block) 與清單。
2. **預覽模式**：點擊編輯框上方的 **「Preview」** 隨時確認學生看到的畫面。

### 步驟 C：設定配分與答案
1. **設定分值**：在「Point Value」輸入該題分數。系統左下角會即時顯示「總配分」。
2. **新增/編輯選項**（選擇題）：
   - 點擊「新增選項」增加內容。
   - 在選項前的圓圈/方框點一下，設定為 **正確答案**。
3. **參考答案**（簡答/問答題）：可填寫給批改老師看的標準答案，學生在考試期間看不到。

## 4. 題目狀態與「凍結」機制 (Freezing)

為了確保考試的公平性，系統設有自動保護機制：

- **未開始狀態**：您可以自由編輯、排序、刪除題目。
- **已開始/有作答狀態**：
  - **禁止修改與刪除**：一旦有學生開始作答（題目被凍結），系統會鎖定現有題目的編輯與刪除權限。
  - **允許新增**：如果您發現少出一題，仍可以點擊「新增題目」補上，新題目會即時出現在學生的考卷中。

---

## 5. 進階：批次處理 (Import/Export)

如果您已經有現成的題庫：
- **匯出 (Export)**：可以匯出 JSON、試題 PDF 或包含正解的 PDF。
- **匯入 (Import)**：點擊「匯入 JSON」，上傳符合規格的 `.json` 檔案。系統會檢查格式並列出預覽，確認後即可完成匯入。

> 詳細 JSON 規格請參考：[Exam JSON 規格說明](/docs/zh-TW/exam-json-import-export)
