# 評測系統說明（現況）

> 文件狀態：2026-02-24

## 評測流程

`提交程式碼 -> 編譯 -> 執行測試 -> 比對輸出 -> 回傳結果`

## 結果代碼

| 代碼 | 說明 |
| --- | --- |
| AC | 全部通過 |
| WA | 輸出錯誤 |
| CE | 編譯失敗 |
| RE | 執行錯誤 |
| TLE | 超時 |
| MLE | 超記憶體 |

## 安全執行模型

- 使用容器隔離執行程式碼
- 限制 CPU/記憶體/程序數等資源
- 關閉非必要權限與網路能力

## 目前注意事項

- 不同語言可能有不同時間係數，請以題目與平台設定為準。
- 若同一份程式在本機與平台結果不同，先確認環境差異與 I/O 格式。
