> 文件狀態：2026-02-24

本指南目標是在 5 分鐘內完成一次題目提交與結果確認。

## 1. 登入

- 支援一般帳密登入與 OAuth/SSO 流程。
- 首次登入請先確認角色（Student/Teacher/Admin）是否正確。

## 2. 進入題目並作答

1. 進入「題目」列表頁
2. 選擇題目並閱讀敘述與限制
3. 於編輯器選擇語言（C/C++/Python/Java）
4. 提交程式碼

## 3. 檢視評測結果

提交後會顯示狀態（如 `AC`、`WA`、`TLE`、`RE`、`CE`），並可查看每次提交細節。

## 4. 若要參加競賽

- 前往「競賽」頁面加入公開或私人競賽。
- 私人競賽需輸入密碼。
- 競賽中可依規則查看排名與提交紀錄。

## 5. 若要使用 AI 助教

- 進入 chatbot 功能頁。
- 系統使用 SSE 串流回覆，可能出現 `thinking`、`tool`、`delta` 等階段性事件。
- 若網路中斷，可重新載入對話 session 後繼續。
