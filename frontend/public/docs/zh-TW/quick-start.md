# 快速入門

本指南將幫助您在 **5 分鐘內**完成第一次程式提交。

## 快速入門清單

按照以下步驟完成您的第一次提交：

| 步驟 | 任務               | 相關說明                                |
| ---- | ------------------ | --------------------------------------- |
| 1    | 登入 QJudge 系統   | [步驟 1：登入系統](#步驟-1登入系統)     |
| 2    | 找到一道適合的題目 | [步驟 2：瀏覽題目](#步驟-2瀏覽題目)     |
| 3    | 撰寫並提交程式碼   | [步驟 3：解題與提交](#步驟-3解題與提交) |
| 4    | 確認評測結果       | [步驟 4：查看結果](#步驟-4查看結果)     |

---

## 步驟 1：登入系統

您可以透過以下方式登入：

| 登入方式     | 說明                        | 建議對象   |
| ------------ | --------------------------- | ---------- |
| **SSO 登入** | 使用 NYCU 帳號直接登入      | NYCU 師生  |
| **一般登入** | 使用註冊的 Email 和密碼登入 | 外部使用者 |

> **首次使用 SSO？** 系統會自動為您建立帳號，無需額外註冊。

---

## 步驟 2：瀏覽題目

登入後，點擊導覽列的「**題目**」進入題目列表頁面。

### 篩選與搜尋功能

| 功能       | 操作說明                   |
| ---------- | -------------------------- |
| 依難度篩選 | 點擊 Easy/Medium/Hard 標籤 |
| 關鍵字搜尋 | 在搜尋框輸入題目名稱或編號 |
| 查看通過率 | 每道題目右側顯示 AC 比率   |

> **新手建議**：從標記為 `Easy` 的題目開始練習。

---

## 步驟 3：解題與提交

```
1. 點擊題目 → 進入詳情頁
        ↓
2. 閱讀題目描述與範例
        ↓
3. 在編輯器撰寫程式碼
        ↓
4. 選擇程式語言 (C/C++/Python/Java)
        ↓
5. 點擊「提交」按鈕
```

### 支援的程式語言

| 語言   | 版本   | 編譯/執行指令        |
| ------ | ------ | -------------------- |
| C      | GCC 11 | `gcc -O2 -std=c11`   |
| C++    | G++ 11 | `g++ -O2 -std=c++17` |
| Python | 3.10   | `python3`            |
| Java   | 17     | `javac` + `java`     |

詳細資訊請參考 [支援的程式語言](/docs/supported-languages)。

---

## 步驟 4：查看結果

提交後，系統會自動評測您的程式碼。結果會在數秒內顯示。

### 評測結果說明

| 結果                  | 縮寫 | 說明             | 下一步行動           |
| --------------------- | ---- | ---------------- | -------------------- |
| Accepted              | AC   | 程式通過所有測試 | 恭喜！挑戰下一題     |
| Wrong Answer          | WA   | 程式輸出錯誤     | 檢查邏輯與邊界條件   |
| Time Limit Exceeded   | TLE  | 程式執行超時     | 優化演算法效率       |
| Memory Limit Exceeded | MLE  | 記憶體超過限制   | 減少資料結構用量     |
| Runtime Error         | RE   | 執行時發生錯誤   | 檢查陣列越界、除零等 |
| Compilation Error     | CE   | 編譯失敗         | 檢查語法錯誤         |

詳細錯誤排解請參考 [常見錯誤](/docs/common-errors)。
