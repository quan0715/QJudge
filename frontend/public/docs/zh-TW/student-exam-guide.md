# 學生考試操作指南

> 文件狀態：2026-03-11 (根據學生版系統操作說明更新)

本指南旨在協助學生熟悉 QJudge 系統的考試流程，包含環境檢查、防作弊機制以及紙筆測驗的操作。

## 1. 基本設定

### 帳號登入
- 進入 [q-judge.com](https://q-judge.com) 官網。
- 點擊「立即登入」，並選擇「使用 SSO 登入」。
- 使用您平常登入 **NYCU Portal** 的學號/人事代號與密碼進行填寫。

### 個人化設置
- 點擊右上角「User」圖示開啟選單。
- **主題設定**：支援淺色 (Light)、深色 (Dark) 及跟隨系統 (System)。
- **語言設定**：提供繁體中文、英文、韓文及日文（建議以繁體中文或英文操作為主）。
- **顯示名稱**：可在設定頁面進一步設置帳號資訊。

## 2. 參加考試流程

1. **進入競賽列表**：點擊導覽列的「競賽」按鈕。
2. **選擇考試**：點擊對應的「競賽/考試」卡片。
3. **註冊/報名**：點擊「報名」或「註冊競賽」按鈕。
   - **私人競賽**：需輸入老師提供的競賽密碼。
   - **匿名模式**：若有開啟，需填寫考場顯示的暱稱。
4. **等候開始**：報名成功後，等候考試時間到達。

## 3. 考前環境與資格檢查 (Pre-check)

在進入防作弊模式的考試前，系統會引導您完成以下步驟：

- **第一步：身份檢查**：確認您的參賽資格與交卷紀錄。
- **第二步：環境檢測**：
  - **瀏覽器**：僅允許最新版 Google Chrome 或 Microsoft Edge（建議使用無痕模式）。
  - **螢幕數量**：必須為單一實體螢幕（請拔除外接顯示器）。
  - **全螢幕模式**：要求在全螢幕狀態下進行作答。
  - **螢幕分享**：必須授權分享「整個螢幕 (Entire Screen)」。
- **第三步：閱讀考試規則**：確認後即可正式進入考試。

## 4. 嚴格考試模式與防作弊偵測

考試過程中，系統會持續偵測以下行為：

### 環境檢查項目
- **全螢幕監控**：若退出全螢幕，系統會發出警告並要求於限期內恢復。
- **螢幕分享**：禁止停止螢幕分享，若中斷未在時限內恢復，將會**自動強制交卷**。
- **單螢幕檢查**：禁止使用多視窗環境。

### 違規行為檢查
- **失焦行動**：包含滑鼠移出視窗、切換分頁 (Tab switching) 或切換應用程式。
- **快捷鍵限制**：系統直接禁用複製貼上及常用危險快捷鍵。

> **警告**：違規行為次數若超過系統設定上限，將會被**強制鎖定**或**直接交卷**，並可能喪失考試資格。

## 5. 線上紙筆測驗操作

進入紙筆測驗（Paper Exam）畫面後：

- **作答方式**：依照考卷內容，直接在作答區輸入文字或選擇選項。
- **顯示模式切換**：
  - **列表模式 (List view)**：一次顯示所有題目。
  - **全題模式 (Full view)**：一次顯示一題，可利用右下/左下按鈕切換上下題。
- **題目導覽**：點擊中間的圖示可縮合/展開題目列表，快速跳轉至指定題目。
- **交卷前確認**：系統會提示尚未作答的題號，確認無誤後點擊「確認交卷」。

## 6. 支援題型說明

- **是非題**：選擇「是 (True)」或「否 (False)」。
- **單選題**：從多個選項中選擇一個最符合的答案。
- **多選題**：勾選所有可能的正確選項。
- **簡答題**：在作答區輸入簡短回答。
- **詳答題/問答題**：在作答區輸入多行詳細回答。

## 建議事項

- 建議使用電腦或筆記型電腦進行考試，避免使用平板或手機。
- 考試前請確保網路連接穩定。
- 若遇到系統異常，請立即連繫現場監考人員。
