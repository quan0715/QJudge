# 參加競賽

了解如何在 QJudge 系統中參加程式競賽。

## 競賽類型

### 公開競賽

所有登入用戶皆可參加的競賽。

### 私人競賽

需要輸入密碼才能參加的競賽，通常用於課程作業或特定活動。

## 參加步驟

### 1. 找到競賽

在「**競賽**」頁面瀏覽可用的競賽列表。您可以看到：

- 競賽名稱與說明
- 開始與結束時間
- 競賽狀態（即將開始 / 進行中 / 已結束）

### 2. 進入競賽

點擊競賽進入競賽頁面。若為私人競賽，需輸入密碼。

### 3. 閱讀規則

競賽開始前，請詳閱競賽規則與計分方式。

### 4. 解題

競賽開始後：

1. 從題目列表選擇題目
2. 閱讀題目描述
3. 撰寫並提交程式碼
4. 查看即時排名

## 計分方式

### ACM/ICPC 模式

- 以解題數量為主要排名依據
- 相同解題數時，依總罰時排名
- 每次錯誤提交增加 20 分鐘罰時

### OI 模式

- 每題依通過的測試點比例給分
- 最終成績為所有題目得分總和

## 競賽期間功能

### 問答 (Clarifications)

若對題目有疑問，可透過問答功能向出題者提問。

### 即時排名

查看自己與其他參賽者的排名狀況。

## 注意事項

**重要提醒**：

- 競賽期間請勿分享程式碼或解題思路
- 請遵守學術誠信規範
- 競賽結束後才能查看他人解答
