# 考試作答統計

> 文件狀態：2026-03-04

本篇說明如何在 QJudge 的 **Exam Statistics** 面板中檢視每道題目的得分分布與選項統計。

## 1. 進入統計面板

在 Admin Dashboard 側邊欄點擊 **Statistics** 圖示即可進入。

> **注意：** 統計面板僅適用於紙筆題考試（Paper Exam）。Coding Test 類型的競賽會顯示「暫時不支援」提示。

## 2. 介面佈局

統計面板採用 **兩欄佈局**：

| 區域 | 內容 |
|------|------|
| **左欄：題目導覽** | 列出所有題目，顯示題號、題目開頭文字、平均分數 |
| **右欄：統計卡片** | 以卡片形式依序顯示每道題目的統計詳情，可上下捲動 |

點擊左欄的題目，右欄會自動捲動到對應的統計卡片。

## 3. 題目統計卡片

每張統計卡片包含以下資訊：

### 標題列

- **題號**（Q1, Q2...）
- **題型標籤**（單選題 / 多選題 / 是非題 / 簡答題 / 問答題）
- **批改進度**（例如「12/15 已批改」，全部批改完成顯示綠色標籤）

### 平均分數

以數字與進度條顯示該題的平均得分與滿分比例。

### 選項分布（客觀題）

適用於單選題、多選題、是非題：

- 每個選項顯示為一列，包含：
  - 選項文字（A, B, C...）
  - 水平長條圖，長度代表該選項被選擇的比例
  - 選擇人數與百分比
  - 正確答案以 ✓ 標記，長條圖顯示為綠色

### 學生作答列表（主觀題）

適用於簡答題、問答題：

- 以手風琴（Accordion）形式列出每位學生的作答
- 標題顯示學生暱稱與得分（或「未批改」）
- 展開後顯示完整的文字作答內容

## 4. 使用建議

- 在批改完所有作答後查看統計，可以獲得最完整的數據
- 客觀題的選項分布有助於分析題目鑑別度——若錯誤選項的選擇率特別高，可能代表題目或選項需要調整
- 主觀題的作答列表可快速檢視學生答題品質，搭配展開 / 收合操作進行瀏覽
