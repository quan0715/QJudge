# 常見錯誤與排查

> 文件狀態：2026-02-24

## 1. 提交後 `CE`（編譯錯誤）

常見原因：

- 語法錯誤
- 使用平台不支援語法或函式庫
- Java 類別名稱非 `Main`

處理方式：

- 先在本地同版本編譯器驗證
- 對照題目限制與輸入輸出格式

## 2. 提交後 `WA`（答案錯誤）

常見原因：

- 輸出格式不一致（空白/換行）
- 邊界條件未處理
- 資料型別溢位

## 3. 提交後 `TLE` / `MLE`

常見原因：

- 演算法複雜度過高
- 記憶體配置過大

處理方式：

- 降低時間複雜度
- 避免不必要的大型中間資料結構

## 4. Chatbot 串流看起來卡住

可能原因：

- 網路短暫中斷
- SSE 串流中途異常

處理方式：

- 重新載入當前 session
- 檢查 backend/ai-service 日誌是否有 `error` 事件

## 5. 開發環境測試失敗但功能正常

常見原因：

- backend 測試使用錯誤 settings（誤連非本機 DB）
- 前端 API 測試缺少測試帳號或 seed data

處理方式：

- 使用 `config.settings.test` + 明確 `DATABASE_URL`
- 補齊測試資料後再重跑
