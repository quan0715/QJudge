## 什麼是 QJudge？

QJudge 是一個專為程式設計教學與練習打造的線上評測系統。它提供了一個完整的環境，讓您可以：

- **練習程式題目** - 瀏覽並解決各種難度的程式題目
- **參加程式競賽** - 在競賽中與其他參賽者一較高下
- **追蹤學習進度** - 查看您的解題統計與歷史紀錄

## 主要功能

### 題目練習

系統提供豐富的題庫，涵蓋不同難度等級：

| 難度   | 說明                             |
| ------ | -------------------------------- |
| Easy   | 適合初學者，基礎語法練習         |
| Medium | 需要一定程式基礎，涉及常見算法   |
| Hard   | 進階題目，需要較深入的演算法知識 |

### 線上競賽

- 即時排名系統
- 支援多種計分模式（ACM/ICPC、OI）
- 競賽期間的問答功能（Clarifications）

### 即時評測

提交程式碼後，系統會自動進行編譯與測試，並即時回報結果。

## 使用者角色與權限

QJudge 支援多種使用者角色，不同角色擁有不同功能：

| 角色    | 可用功能                         | 詳細說明                               |
| ------- | -------------------------------- | -------------------------------------- |
| Student | 瀏覽題目、提交程式、參加競賽     | 預設角色                               |
| Teacher | 建立競賽、管理題目、查看學生成績 | [查看教師指南](/docs/teacher-overview) |
| Admin   | 管理使用者、系統設定、權限控制   | [查看管理員指南](/docs/admin-overview) |

> **提示**：如需升級為教師或管理員角色，請聯繫系統管理員。
