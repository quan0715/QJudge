# 教師功能總覽（現況）

> 文件狀態：2026-02-24

## 教師主要權限

- 建立與管理競賽
- 管理題目與測試資料
- 檢視學生提交與成績
- 使用公告與課程流程工具

## 競賽與題目工作流

1. 建立競賽（公開/私人）
2. 設定時間、計分規則、參與限制
3. 匯入或建立題目（可搭配 YAML 匯入）
4. 發布競賽並監控提交

## Exam V2（目前進度）

目前已具備流程骨架：

- 考生註冊
- 考前檢查
- 作答流程
- 提交檢查
- 評分與結果頁

> 提醒：Exam V2 目前屬持續收斂階段，請在正式開課前先做小規模演練。

## 教學實務建議

- 先用小型測驗驗證題目與測試資料品質。
- 競賽前先確認題目可見性與語言限制。
- 對高風險題目準備備援說明與公告模板。
