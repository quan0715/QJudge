## 提交流程

1. 選擇題目
2. 選擇語言
3. 撰寫程式碼
4. 點擊提交
5. 檢查評測結果與執行資訊

## 支援語言

目前主流程支援：

- C
- C++
- Python
- Java

詳細說明請見 [/docs/supported-languages](/docs/supported-languages)。

## 結果狀態

| 狀態 | 意義 |
| --- | --- |
| AC | 全部測試通過 |
| WA | 輸出不正確 |
| TLE | 超過時間限制 |
| MLE | 超過記憶體限制 |
| RE | 執行錯誤 |
| CE | 編譯錯誤 |

## 實務建議

- 嚴格比對輸出格式（空白、換行、大小寫）。
- 優先處理邊界條件（空輸入、最大值、負值）。
- 若本地可過但平台失敗，先檢查 I/O 與資料範圍假設。
